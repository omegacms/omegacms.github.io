\Omega\Filesystem\AdapterLocalAdapter

Local filesystem driver.

The LocalAdapter class represents a filesystem adapter designed for local storage using Flysystem. It extends the abstract filesystem adapter, providing specific implementation details to connect to the local filesystem using Flysystem's LocalFilesystemAdapter.

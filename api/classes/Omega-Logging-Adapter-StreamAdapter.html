\Omega\Logging\AdapterStreamAdapter

Stream adapter class.

The StreamAdapter class is a concrete implementation of the abstract logging adapter, providing logging functionality to a stream. It utilizes the Monolog library for handling log messages and supports logging informational, warning, and error messages.

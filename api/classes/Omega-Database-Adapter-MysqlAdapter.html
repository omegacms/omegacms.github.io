\Omega\Database\AdapterMysqlAdapter

MySQL adapter class.

The MysqleDatabaseAdapter class is an implementation of the abstract AbstractDatabaseAdapter and is specifically tailored for MySQL database connections. This adapter provides mysql-specific database management features while inheriting the common database functionality defined in the parent class.
